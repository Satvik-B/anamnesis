# Full Example

This example shows a complete claude-memory setup with all directories
and files populated.

To generate this structure in your own project, run:

```bash
claude-memory init
```

Or copy the files from `skeleton/` in the claude-memory repository:

```
skeleton/
  rules/memory-rule.md      -> .claude/rules/memory-rule.md
  memory/MEMORY.md           -> .claude/memory/MEMORY.md
  memory/INDEX.md            -> .claude/memory/INDEX.md
  memory/archive/            -> .claude/memory/archive/
  memory/slack/              -> .claude/memory/slack/
  skills/friday/             -> .claude/skills/friday/
```

See the [Getting Started guide](../../docs/getting-started.md) for a detailed walkthrough.
